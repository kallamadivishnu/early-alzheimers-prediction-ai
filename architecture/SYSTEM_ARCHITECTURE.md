# System Architecture

## Cloud-Based Early Alzheimer's Disease Prediction System

### Workflow

MRI Brain Images
        ↓
Google Drive Storage
        ↓
Google Colab Processing
        ↓
Data Preprocessing
        ↓
CNN Model Training
        ↓
Model Evaluation
        ↓
Disease Prediction
        ↓
Result Visualization
        ↓
GitHub Documentation

## Components

### Data Layer
- OASIS-1 MRI Dataset
- Brain MRI Images

### Storage Layer
- Google Drive
- Cloud Storage for datasets and models

### Processing Layer
- Google Colab
- Python
- TensorFlow
- Keras

### Machine Learning Layer
- Convolutional Neural Network (CNN)
- Feature Extraction
- Classification

### Output Layer
- Alzheimer's Disease Prediction
- Accuracy Metrics
- Visual Results
