# Cloud-Based Early Alzheimer's Disease Prediction System

## Domain

Cloud Computing

## Project Overview

This project presents a cloud-based Alzheimer's Disease prediction system that uses MRI brain scans and Deep Learning techniques for early disease detection.

The system utilizes cloud resources provided by Google Colab for model training and Google Drive for cloud storage. MRI images from the OASIS-1 dataset are processed and analyzed using a Convolutional Neural Network (CNN) to identify patterns associated with Alzheimer's Disease.

The project demonstrates how cloud computing can support scalable medical data processing, centralized storage, remote accessibility, and AI-based healthcare analytics.

---

## Problem Statement

Traditional healthcare systems often face challenges such as delayed diagnosis, isolated medical analysis, limited scalability, and lack of continuous monitoring.

This project aims to leverage cloud computing and artificial intelligence to assist in the early prediction of Alzheimer's Disease using MRI image analysis.
