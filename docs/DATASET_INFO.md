# Dataset Information

## Dataset Name
OASIS-1 Brain MRI Dataset

## Description
The Open Access Series of Imaging Studies (OASIS) dataset provides MRI brain images for Alzheimer's Disease research.

## Data Type
- MRI Brain Scans
- Medical Imaging Data

## Purpose
Used for training and testing the CNN model for Alzheimer's Disease prediction.

## Dataset Features
- Brain MRI Images
- Multiple Subjects
- Alzheimer's Related Imaging Data

## Source
OASIS Brain Imaging Project

## Usage in Project
The dataset is used for:
1. Image preprocessing
2. Model training
3. Model validation
4. Disease prediction

## Cloud Storage
Dataset stored using Google Drive and processed through Google Colab.
