# Cloud-Based Early Alzheimer's Disease Prediction System

## Author
Kallamadi Vishnukanth Reddy

## Domain
Cloud Computing

## Abstract

This project presents a cloud-based system for the early prediction of Alzheimer's Disease using MRI brain scan images and Deep Learning techniques. The system leverages cloud resources for storage, processing, and model training. MRI images from the OASIS dataset are analyzed using a Convolutional Neural Network (CNN) model to identify patterns associated with Alzheimer's Disease.

The project demonstrates how cloud computing platforms can support medical image analysis by providing scalable storage, remote accessibility, and computational resources.

## Objectives

- Early detection of Alzheimer's Disease.
- Utilize cloud-based resources for processing and storage.
- Implement Deep Learning for MRI image classification.
- Demonstrate practical application of Cloud Computing in healthcare.
